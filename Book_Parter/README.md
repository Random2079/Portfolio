# Book_Parter

Мелкие утилиты для чистки текста книг (после Epub2txt и т.п.).

## Скрипты

| Файл | Зачем |
|------|--------|
| `remove_empty_lines.py` | Убрать пустые строки из TXT |
| `demiurge_worker_test.py` | Черновик/тест воркера (эксперимент) |

## Запуск

```powershell
python Book_Parter/remove_empty_lines.py
```

Зависимости сверх стандартной библиотеки Python не нужны (для `remove_empty_lines`).
