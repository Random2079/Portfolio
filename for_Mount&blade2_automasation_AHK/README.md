# Mount & Blade II — макросы AHK

Автоматизация в игре через **AutoHotkey v1**.

## Запуск

1. Установи [AutoHotkey v1](https://www.autohotkey.com/).
2. Открой `AutoHotkey.ahk` (двойной клик или через AHK).

## Управление

| Клавиша | Действие |
|---------|----------|
| Боковая кнопка мыши 1 (`XButton1`) | Старт / пауза крафта (forge loop) |
| Клавиша `1` | Автоклик вкл/выкл |
| Боковая кнопка мыши 2 (`XButton2`) | Полностью закрыть скрипт |

Крафт и автоклик **не работают одновременно** — включение одного гасит другой.

## Файлы

- `AutoHotkey.ahk` — основной скрипт  
- `checker.ahk` — вспомогательный  
- Отдельный `smelt_clicker.ahk` удалён: автоклик влит в основной файл
