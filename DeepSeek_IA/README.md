# DeepSeek_IA

Примеры запросов к **DeepSeek API** через официальный OpenAI-совместимый клиент.

## Настройка

В корне `DS_Projects` файл `.env` (не в git):

```
DEEPSEEK_API_KEY=твой_ключ
```

## Запуск

```powershell
pip install openai python-dotenv
python DeepSeek_IA/deepseek_python_20260527_8e582e.py
```

## Зависимости

- `openai`
- `python-dotenv`
