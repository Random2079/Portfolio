# sandbox_ui

Песочница для GUI-библиотек. **Не** боевой YouTube/TTS.

## Кандидаты (одинаковая морда: ссылка + Скачать + статус)

| # | Демо | Стек | Запуск |
|---|------|------|--------|
| 1 | `demo_customtkinter.py` | CustomTkinter | `python sandbox_ui/demo_customtkinter.py` |
| 2 | `demo_pyqt_qss.py` | PyQt5 + QSS (уже в проекте) | `python sandbox_ui/demo_pyqt_qss.py` |
| 3 | `demo_flet.py` | Flet | `python sandbox_ui/demo_flet.py` |

React / веб — не сюда (скорее новостник IDEA-003).

## Зависимости песочницы

```powershell
pip install customtkinter flet
# PyQt5 у тебя уже есть
```

Открой **два-три окна рядом**, потыкай, скажи какой вайб ближе — потом переносим в YouTube_Translator.
