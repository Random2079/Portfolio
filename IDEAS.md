# Памятка идей

Общая база идей для `DS_Projects`. ИИ сверяется с этим файлом и обновляет его по просьбе («добавь в памятку», «отметь сделанным» и т.п.). Не удалять и не менять смысл идей без подтверждения.

Планы «как делать» (MVP-шаги): папка [`.cursor/plans/`](.cursor/plans/) — оглавление в [`памятка-идей.md`](.cursor/plans/памятка-идей.md).

## Как пользоваться

- Новая мысль → раздел **Входящие**
- Решили делать → **Запланировано**
- Уже копаем → **В работе**
- Сделано → **Готово**
- Не актуально → **Отложено / отклонено**

Формат записи:

```
### ID — короткий заголовок
- Проект/область:
- Дата:
- Статус:
- Описание:
- Следующие шаги:
```
