# План: памятка идей для DS_Projects

Видимая копия плана в папке проекта (служебный Accept/Reject-план Cursor живёт отдельно у Cursor — это нормально; здесь лежит то, что видно в Explorer).

## Цель

Общая база идей + правило, чтобы ИИ не «забывал» между чатами.

## Шаги

1. Создать корневой [`IDEAS.md`](../../IDEAS.md) со структурой статусов и стартовым набором идей.
2. Создать правило [`.cursor/rules/ideas-memory.mdc`](../rules/ideas-memory.mdc) (`alwaysApply: true`).
3. Держать этот файл в `.cursor/plans/` как заметку по плану в репозитории.

## Идеи в `IDEAS.md`

1. AI-агент с TTS (как Neuro-sama)
2. TTS для чата Cursor
3. Новостник портфеля (тикеры → новости → уведомления)
4. Макросы Mount & Blade II (авто-клик + авто-создание оружия)
5. Авто-мут в Discord (порог ~60 дБ → mute)
6. Научиться Kali Linux
7. Отслеживатель профилей Discord
8. Личная защита капитала и устройств
