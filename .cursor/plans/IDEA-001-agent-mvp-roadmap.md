# IDEA-001 — AI-агент: тонкий MVP-roadmap

## Цель

Не строить Neuro целиком. Собрать понимание из кирпичей; кодить агента только крошечным MVP позже.

## Карта кирпичей

| Кусок | Смысл | Где сейчас |
|-------|--------|------------|
| Рот | TTS | Cursor_TTS (IDEA-002) |
| Уши | уровень / мут | IDEA-005 |
| Мозг | LLM | IDEA-009 GGUF / API |
| Руки | клики / окна | AHK / позже whitelist |
| Морда | UI | отдельно; веб скорее к новостнику |

## Порядок (не параллельно всё)

1. Стабильный рот (002 Piper/local).  
2. Уши MVP (005 слой A → B).  
3. Мозг: один чат-цикл (локальный GGUF или API) без рук.  
4. Склеить: ответ модели → TTS.  
5. Руки — только 1–2 разрешённых действия, не весь Windows.

## Не в scope сейчас

- Полноценная «персона» как Neuro  
- Автономный кликер по всему столу  
- Красивый стрим-UI

## Готово для этого документа

Порядок ясен; каждая следующая сессия берёт **один** кирпич из списка выше, не «агент целиком».
