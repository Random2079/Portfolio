# IDEA-005 — Авто-мут Discord («уши»)

## Цель

Если громкость на входе выше порога (~60 дБ) — мьютим Discord (или сначала логируем MUTE). Кирпич к агенту, не весь агент.

## Не в scope

- Красивый UI (макс. консоль / крошечное окно)  
- «Руки» агента (клики по столу)  
- Учим чужие аккаунты

## MVP по слоям

### A — только уши (сначала это)

1. Скрипт читает уровень микрофона (например `sounddevice` / `pyaudio`).  
2. Считает приближённый уровень, сравнивает с порогом.  
3. Печатает `OK` / `LOUD` в консоль.  
4. Папка: например `Discord_AutoMute/` или `sandbox_ears/`.

### B — mute в Discord

5. Уточнить способ: хоткей Discord / Windows mute / API (что проще на твоей машине).  
6. По `LOUD` → mute; правило unmute (тишина N сек / ручной).  
7. Конфиг: порог, устройство входа, задержка.
