# IDEA-011 — Субтитры с таймкодами (не каждое слово)

## Цель

Текст с привязкой ко времени (фразы / куски), чтобы сопоставить момент с картинкой в ролике. Не word-level на каждую секунду.

## Не в scope

- Полный редизайн UI (это IDEA-010)  
- Свой плеер видео обязателен в MVP

## Шаги

1. Посмотреть текущий вывод [`YouTube_Translator/Subtitle_App.py`](../../YouTube_Translator/Subtitle_App.py) и whisper/HTML-путь.  
2. Выбрать формат MVP: SRT (блоки) или TXT вида `[mm:ss] фраза`.  
3. Правило склейки: по паузе / по длине / по предложениям (зафиксировать одно).  
4. Реализовать пост-обработку сегментов → файл рядом с субтитрами.  
5. Короткий тест на одном ролике «с картинками».  
6. Строка в README YouTube_Translator.

## Готово когда

Есть файл с таймкодами по фразам; глазами можно прыгнуть к моменту на видео.
